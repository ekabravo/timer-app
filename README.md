# Joyful Visual Timer

Joyful Visual Timer is heavily inspired by the [Not Boring Apps Timer](https://notbor.ing/product/timer). I wanted a web version that I could use on my laptop and mobile in a way I want to.

One important difference is that this timer is intentionally silent. I found it frustrating when timer alarms interrupted Automatic Switching between Apple devices, so this app focuses on a clear visual countdown instead of sound.

This app is intentional foreground visual timer PWA.

## How to use it

Open the deployed timer from the repository description, then add it to your home screen as a web app. This caches the assets properly, lets you use it without the browser interface, and keeps it available offline.

For desktop use add deployed version to bookmarks. Apparantly I will add a custom domain.
